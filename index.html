使用教程

详细的使用指南，助您快速上手

重要提示

菜单适用于 GTA5 增强版，正常开启 BattlEye 使用

菜单免费使用，无法承担封禁风险，自行删除即可

禁止使用双开或多开菜单，否则后果自负

建议不要使用搜狗输入法以及搜狗的其他产品

默认键位

小键盘 - 显示/隐藏菜单
小键盘 8246 控制上下左右
小键盘 5 确认
小键盘 0 返回

快捷键

F3 跳过动画
F4 向前闪现
F5 传送到导航点/任务点

操作键位/热键可根据需要在配置文件内自行修改

启动流程

  1. 关闭内核隔离

    关闭 Windows 系统的内核隔离，然后重启电脑

  2. 关闭反作弊

    卸载或关闭开机预启动的反作弊（如 Vanguard、ACE 等）

  3. 解压文件

    解压下载的压缩文件

  4. 驱动加载

    在启动游戏前先运行一次程序（.exe 文件，随机进程名防检测）进行驱动加载，显示加载成功后关闭命令窗口，然后启动游戏（GTA5 增强版）

  5. 启动菜单

    打开游戏后进入到故事模式，再次运行程序，启动菜单

  6. 安装中文

    菜单默认为英文，双击运行 install_chinese.bat 即可安装中文语言文件

驱动加载说明

驱动加载成功

命令窗口显示：[+] Operation Completed Successfully

驱动加载失败

命令窗口显示：Failed To Load Driver

可能是下载微软符号文件失败，加速微软相关或者科学上网后重试，或者重启电脑

驱动正在加载

命令窗口显示：Loading KD..

正在下载文件，需要耐心等待，直至加载成功

驱动无法加载

命令窗口显示：Error: Unsupported Windows Version

当前的 Windows 系统不支持使用

如果驱动加载部分出现各种问题请自行排查解决，只有加载成功才能正常使用

Error Code: 536870921 并不是错误，只是检测 GTA 相关进程，按启动流程的步骤进行即可

配置目录

菜单配置目录： %UserProfile%\AppData\Roaming\Alice2333

Lua 文件目录： %UserProfile%\AppData\Roaming\Alice2333\lua_scripts

Lua 文件需要放到 lua_scripts 目录下
